# 🐾 PetQR – Cómo funciona

PetQR es un sistema de identificación inteligente para mascotas mediante código QR.

## 📱 Funcionamiento
1. El QR está en la chapa del collar.
2. Se escanea con cualquier celular.
3. Se abre una página web.
4. Se puede llamar o enviar ubicación por WhatsApp.

## 🏷️ Incluye
- 2 chapas por mascota
- 1 grande y 1 chica
- Ambas al mismo precio

## 🚀 Tecnología
HTML + CSS + JavaScript  
No requiere apps ni registros.
